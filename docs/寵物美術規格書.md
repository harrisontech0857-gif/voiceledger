# 語記 (VoiceLedger) — 寵物美術資源規格書

> 本文件為 AI 生圖工具（Gemini Imagen / ImageFX）的 prompt 參考規格。
> 所有圖片用於 Flutter App 內的寵物養成系統。

---

## 一、角色設定

| 項目 | 說明 |
|------|------|
| 種族名稱 | 招財貓（Moneycat） |
| 設計風格 | 日式 Q 版 × 台灣元素，圓潤可愛，略帶手繪質感 |
| 品牌色 | 主色 #E07C4F（暖橘），輔色 #FFD700（金），背景透明 |
| 情緒表現 | 透過眼睛大小、嘴型、耳朵角度、尾巴姿態傳達 |
| 一致性 | 所有階段共用相同的圓眼、短鼻、圓臉基因，進化只加細節不改比例 |

## 二、進化階段（5 階段）

### Stage 1：神秘蛋 (egg)
- **外觀**：橢圓金蛋，表面有淡淡的貓掌印花紋和一條裂縫發光
- **大小感**：最小，約佔畫面 60%
- **動態暗示**：微微搖晃、裂縫處有金色光芒
- **Prompt 參考**：`A cute golden egg with subtle cat paw prints on the surface, a glowing crack on top emitting warm orange light, transparent background, flat illustration style, kawaii, 2D game asset`

### Stage 2：幼貓 (baby)
- **外觀**：剛破殼的小奶貓，大頭小身體，橘白配色，戴著蛋殼帽
- **特徵**：水汪汪大眼、粉紅小鼻子、短短的尾巴、頭上頂著半個蛋殼
- **大小感**：比蛋大一點，佔畫面 70%
- **Prompt 參考**：`A tiny adorable orange and white kitten with oversized round eyes, wearing half an eggshell as a hat, chibi proportions, big head small body, transparent background, flat kawaii illustration, 2D game asset`

### Stage 3：少年貓 (teen)
- **外觀**：長大一些的橘貓，開始有招財貓的特徵，一隻手微微舉起
- **特徵**：更有精神的眼神、脖子繫著金鈴鐺、尾巴有金色尖端
- **大小感**：佔畫面 80%
- **Prompt 參考**：`A young orange tabby cat in chibi style, one paw slightly raised like a maneki-neko, wearing a gold bell on a red collar, golden tail tip, confident cute expression, transparent background, flat kawaii illustration, 2D game asset`

### Stage 4：招財貓 (adult)
- **外觀**：完整的招財貓形態，右手高舉招財，左手抱著金幣
- **特徵**：典型招財貓姿勢但 Q 版化，紅色圍兜上繡「語」字，金幣上有 NT$ 符號
- **大小感**：佔畫面 90%
- **Prompt 參考**：`A cute chibi maneki-neko (lucky cat) in orange and white, right paw raised high beckoning, left paw holding a gold coin with NT$ symbol, wearing a red bib with Chinese character 語, golden bell, happy expression, transparent background, flat kawaii illustration, 2D game asset`

### Stage 5：金財神貓 (master)
- **外觀**：華麗進化版，全身散發金光，頭戴小皇冠，身旁飄著金幣
- **特徵**：金色光環、迷你皇冠、華麗的紅金披風、周圍有 3-5 顆飄浮金幣和星星特效
- **大小感**：最大，佔滿畫面
- **Prompt 參考**：`A majestic chibi golden lucky cat wearing a tiny crown and a red-gold cape, glowing golden aura, floating gold coins and sparkles around it, radiating prosperity, transparent background, flat kawaii illustration, premium 2D game asset`

## 三、心情表情（每階段 4 種）

> 蛋階段只需 1 張（固定外觀），其他 4 階段各需 4 種表情 = **17 張**

| 心情 | 表情描述 | 眼睛 | 嘴巴 | 附加 |
|------|---------|------|------|------|
| happy | 開心閃亮 | ★ 星星眼或笑成彎月 | 大笑 w 嘴 | 頭頂冒愛心或星星 |
| neutral | 日常待機 | 正常圓眼 | 微笑 | 無 |
| hungry | 餓了想吃 | 淚汪汪水潤眼 | 癟嘴或口水 | 頭頂冒「…」氣泡 |
| sleepy | 想睡了 | 半閉眼或 Z 字 | 打哈欠 | 頭頂冒 Zzz |

## 四、圖片技術規格

| 項目 | 規格 |
|------|------|
| 格式 | PNG（透明背景） |
| 解析度 | 512×512 px（@2x），1024×1024 px（@3x） |
| 色彩空間 | sRGB |
| 背景 | 透明 (alpha channel) |
| 風格一致性 | 同一批次使用相同 seed / style reference |
| 邊距 | 角色四周至少留 10% 空白（避免裁切） |

## 六、在 Flutter 中的使用方式

```dart
/// 根據寵物狀態取得對應圖片路徑
String getPetImagePath(PetStage stage, PetMood mood) {
  if (stage == PetStage.egg) return 'assets/images/pet/egg.png';
  final stageName = stage.name;   // baby, teen, adult, master
  final moodName = mood.name;     // happy, neutral, hungry, sleepy
  return 'assets/images/pet/${stageName}_${moodName}.png';
}

/// Widget 使用
Image.asset(
  getPetImagePath(pet.stage, pet.mood),
  width: 72,
  height: 72,
  fit: BoxFit.contain,
)
```

## 七、Gemini Prompt 範本（批次生成）

把以下 prompt 貼給 Gemini ImageFX / Imagen 3：

### 風格統一 prompt prefix（每張都加）
```
Style: flat kawaii 2D game character illustration, chibi proportions,
soft rounded shapes, clean outlines, warm color palette (orange #E07C4F,
gold #FFD700, white, red accents), transparent background, no shadow,
consistent character design across all variations, suitable for mobile app UI.
```

### 批次生成指令
```
Generate a character sheet for a lucky cat (maneki-neko) game mascot
showing 5 evolution stages from left to right:

1. Golden egg with cat paw prints and glowing crack
2. Tiny baby kitten with eggshell hat, big round eyes
3. Young cat with gold bell collar, one paw slightly raised
4. Full maneki-neko with raised paw, holding gold coin marked "NT$",
   red bib with character "語"
5. Majestic golden version with crown, cape, floating coins and sparkles

All stages share the same orange-white color scheme and round face shape.
Transparent background. Flat kawaii style. Each character should be clearly
separated.
```

## 九、驗收標準

- [ ] 所有 17 張圖片尺寸一致（512×512 或 1024×1024）
- [ ] 透明背景，無白邊殘留
- [ ] 5 個階段的角色外觀有明確差異但風格統一
- [ ] 4 種心情表情一眼可區分
- [ ] 品牌色（暖橘 #E07C4F）出現在每個階段
- [ ] 在 72×72 dp 縮圖下仍清晰可辨
- [ ] 檔案命名符合上述規則
