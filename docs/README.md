### Q: 文檔有多詳細？
A: 非常詳細。8,900+ 行文檔涵蓋了從高層架構到逐行代碼的所有內容。

### Q: 我應該先讀哪個文件？
A: 根據你的角色：
- 想快速了解 → `交付物清單.md`
- 想理解架構 → `實作架構概覽.md`
- 想開發 API → `API端點設計.md`

### Q: 代碼在哪裡？
A: Edge Functions 代碼在 `/backend/supabase/functions/` 目錄下：
- `voice-entry/index.ts`
- `passive-inference/index.ts`
- `ai-chat/index.ts`
- `daily-wisdom/index.ts`

3. 部署 Edge Functions
