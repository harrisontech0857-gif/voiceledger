## 快速查詢模板

### 用戶月度統計
```sql
SELECT * FROM user_monthly_stats
WHERE user_id = 'user-id'
ORDER BY month DESC
LIMIT 12;
```

### 預算使用率
```sql
SELECT * FROM budget_utilization
WHERE user_id = 'user-id' AND status IN ('warning', 'exceeded');
```

### 分類支出分布
```sql
SELECT * FROM category_spending_stats
WHERE user_id = 'user-id'
ORDER BY total_amount DESC;
```

### 用戶當月交易
```sql
SELECT * FROM transactions
WHERE user_id = 'user-id'
  AND transaction_date >= '2026-03-01'
  AND transaction_date < '2026-04-01'
ORDER BY transaction_date DESC;
```

### 家庭成員結算
```sql
SELECT * FROM family_settlements
WHERE ledger_id = 'ledger-id'
  AND status = 'pending'
ORDER BY created_at;
```
