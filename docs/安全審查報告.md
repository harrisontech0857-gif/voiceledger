# 安全審查報告

## 概述
本文件提供 VoiceLedger 應用的安全性審查檢查清單，涵蓋 RLS 政策、環境變數、認證回調和敏感資料處理。

## 1. RLS（行級安全性）政策檢查清單

### 用戶表 (users)
- [ ] 確認用戶只能檢視自己的個人資料
- [ ] 檢查是否啟用 `auth.uid()` 檢驗
- [ ] 驗證不能修改其他用戶的資料
- [ ] 確認刪除操作受限於用戶所有者

### 交易表 (transactions)
- [ ] 確認用戶只能檢視自己的交易記錄
- [ ] 檢查 RLS 政策中是否正確使用 `auth.uid()`
- [ ] 驗證新增交易時自動關聯到當前用戶
- [ ] 確認用戶不能修改他人的交易記錄
- [ ] 檢查刪除操作是否受到限制

### 預算表 (budgets)
- [ ] 確認只有所有者能檢視預算設置
- [ ] 檢查編輯和刪除操作是否受限
- [ ] 驗證共享預算的訪問控制（如適用）

### 訂閱表 (subscriptions)
- [ ] 確認用戶只能檢視自己的訂閱狀態
- [ ] 驗證不能修改他人的訂閱資訊

## 2. 環境變數安全性

### .env 檔案管理
- [ ] 確認 `.env` 文件已加入 `.gitignore`
- [ ] 驗證不將敏感資訊提交至版本控制
- [ ] 檢查所有環境變數都有安全的預設值

### 必需環境變數
```
SUPABASE_URL          # Supabase 專案 URL
SUPABASE_ANON_KEY     # 可公開的匿名密鑰
SUPABASE_SERVICE_KEY  # 僅用於後端的服務角色密鑰（絕不公開）
```

### 敏感資訊保護
- [ ] 服務角色密鑰僅存放於安全的後端環境
- [ ] 不在客戶端代碼中硬編碼任何密鑰
- [ ] 匿名密鑰中刪除不必要的權限
- [ ] 定期輪換密鑰

## 3. 認證回調安全性

### OAuth/SSO 回調 URL
- [ ] 確認回調 URL 在 Supabase 控制台正確配置
- [ ] 驗證 URL 使用 HTTPS（生產環境）
- [ ] 檢查回調中驗證 CSRF 令牌
- [ ] 確認不將敏感資訊放在 URL 參數中

### 認證流程
- [ ] 驗證用戶登錄後接收安全的 JWT 令牌
- [ ] 檢查令牌存儲在安全的存儲中（不是 localStorage）
- [ ] 確認會話超時機制已實現
- [ ] 驗證登出時正確清除令牌

### 密碼安全
- [ ] 密碼符合最小強度要求
- [ ] 檢查是否使用 HTTPS 傳輸密碼
- [ ] 驗證密碼重置郵件的安全性
- [ ] 確認重置令牌有過期時間

## 4. 敏感資料處理

### 個人可識別信息 (PII)
- [ ] 加密存儲敏感個人資料（例如銀行帳號）
- [ ] 檢查不在日誌中記錄 PII
- [ ] 驗證備份和日誌文件的加密
- [ ] 確認 GDPR/隱私法規遵從

### 金融資料
- [ ] 所有交易數據使用 HTTPS 加密傳輸
- [ ] 檢查敏感金融信息是否在客戶端被掩蓋
- [ ] 驗證不存儲完整的銀行帳號或信用卡信息
- [ ] 確認符合 PCI DSS 標準

### 位置資料
- [ ] 檢查用戶是否同意收集位置資訊
- [ ] 驗證位置資料的加密存儲
- [ ] 確認用戶能夠選擇退出位置跟蹤
- [ ] 檢查位置資料的最小化處理

### 語音數據
- [ ] 驗證語音記錄的加密傳輸
- [ ] 檢查語音數據存儲的安全性
- [ ] 確認用戶能夠刪除語音記錄
- [ ] 檢查第三方 API（如語音識別）的隱私政策

## 5. 通訊和日誌安全

### 日誌管理
- [ ] 確認日誌不包含敏感資訊（密碼、令牌、PII）
- [ ] 驗證日誌文件的訪問控制
- [ ] 檢查定期日誌輪換和存檔
- [ ] 確認日誌加密存儲

### API 通訊
- [ ] 驗證所有 API 調用使用 HTTPS
- [ ] 檢查 API 速率限制和 DoS 防護
- [ ] 確認 API 金鑰不在 URL 中傳輸
- [ ] 驗證 API 請求頭包含必要的安全標頭

## 6. 第三方集成安全

### Supabase
- [ ] 定期審查 Supabase 安全設置
- [ ] 檢查連接池配置
- [ ] 驗證備份和恢復過程
- [ ] 確認監控和告警已配置

### OpenAI/AI 服務
- [ ] 驗證 API 密鑰的安全存儲
- [ ] 檢查不發送 PII 給 AI 服務
- [ ] 審查 AI 服務的隱私政策
- [ ] 確認符合資料使用條款

## 7. 應用層安全

### Flutter 應用
- [ ] 檢查應用是否在 AndroidManifest.xml 中使用 cleartext traffic 政策
- [ ] 驗證不將敏感資訊存儲在 SharedPreferences（未加密）
- [ ] 檢查使用 flutter_secure_storage 存儲敏感資訊
- [ ] 確認應用簽名密鑰的安全保管

### 代碼安全
- [ ] 掃描代碼中的硬編碼密鑰
- [ ] 驗證不使用已棄用的加密算法
- [ ] 檢查序列化和反序列化安全
- [ ] 確認驗證和授權邏輯正確實現
