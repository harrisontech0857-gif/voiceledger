# VoiceLedger 語記

> **不是記帳 App，是你口袋裡的 AI 財務秘書。**
> 她認識你、了解你的生活節奏、在你忘記時默默幫你記上、在你需要時給出貼心建議。

## 為什麼不一樣？

傳統記帳 App 是「工具」—— 你不打開它就沒用。
語記是「秘書」—— 她主動工作，越用越懂你。

## 十大 AI 特色

| # | 特色 | 說明 |
|---|------|------|
| 1 | 語境感知多筆記帳 | 一次說完多筆，AI 自動拆分分類 |
| 2 | AI 財務教練 | 主動分析消費模式，給可執行建議 |
| 3 | 對話式帳本查詢 | 用自然語言問帳，AI 即時回答+趨勢 |
| 4 | 智能發票 OCR | 拍照+語音混合輸入 |
| 5 | 消費預測與異常偵測 | 預測月底支出，大額消費即時提醒 |
| 6 | **被動式半自動記帳** | GPS定位+照片+銀行通知+習慣推斷 |
| 7 | **每日金句與收藏牆** | 個人化財務洞察，每天一個打開理由 |
| 8 | **AI 財務秘書人格** | 認識你、懂你節奏、情境式對話 |
| 9 | **智能記帳補完** | 週末回顧遺漏，多線索交叉比對 |
| 10 | **AI 生活記帳日記** | 自動生成每日摘要，年度財務報告 |

## 技術棧

| 層級 | 技術 |
|------|------|
| 前端 | Flutter 3.x + Dart + Riverpod |
| 後端 | Supabase (PostgreSQL + Auth + Edge Functions) |
| AI | OpenAI Whisper + Claude/GPT Vision + 多模態推理 |
| 感知 | Core Location / Geofencing + Photos Framework + NotificationListener |
| 自動化 | n8n + GitHub Actions |
| 付費 | RevenueCat |

## AI 五層執行架構

```
感知層    →  語音 / GPS / 相簿 / 推播 / 行事曆
融合推理層 →  多模態理解 → 消費事件推斷 → 信心分數
互動層    →  高信心自動記 / 中信心詢問 / 低信心週末回顧
人格層    →  個人檔案 + 生活節奏 + 情緒感知 → 個人化對話
價值層    →  記錄 → 分析 → 洞察 → 引導 → 行為改變
```

## 開發流程

**OMC 五層生命週期** + **三重審查機制**

```
L1 需求策略 → L2 架構設計 → L3 開發實作 → L4 交付部署 → L5 營運監控
     ↓              ↓              ↓              ↓              ↓
  [R1+R2+R3]    [R1+R2+R3]    [R1+R2+R3]    [R1+R2+R3]    [R1+R2+R3]
```
